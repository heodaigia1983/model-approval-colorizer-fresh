MODEL APPROVAL COLORIZER - FRESH PACKAGE

Mục đích:
- Tạo một repo mới hoàn toàn để test
- Dùng link mới, tránh cache và nhầm với repo cũ

Cách dùng:
1. Tạo repo GitHub mới, ví dụ:
   model-approval-colorizer-fresh

2. Upload 4 file:
   - manifest.json
   - index.html
   - app.js
   - icon.svg

3. Sửa file manifest.json:
   thay YOUR_GITHUB_USERNAME và YOUR_REPO_NAME bằng repo mới của anh

4. Bật GitHub Pages:
   Settings -> Pages -> Deploy from a branch -> main -> /(root)

5. Link plugin sẽ là:
   https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPO_NAME/manifest.json

6. Vào Trimble:
   Settings -> Extensions -> Plugin URL -> dán link trên -> Add

Gợi ý:
- Nếu repo cũ là trimble-extension
- repo mới có thể đặt:
  model-approval-colorizer-fresh
